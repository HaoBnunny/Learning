C compilers in the modern era are not just 'compilers', these "compilers" contain all the other stages of building an application too.
Aside from the stages, Modern compilers are also built out of components rather just being 'Compilers':
- Front-end of a compiler:
  Includes a parser & outputs the intermediate code.
- Back-end of a compiler:
  Optimises the intermediate code & outputs machine code.
