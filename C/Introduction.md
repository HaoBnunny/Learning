# C the programming language
Made - A long time ago, it doesn't matter.
Low-level, giving you the most control you can have while using a programming language (barring pure assembly)
Used as a base for **close to everything** including even other programming languages, i.e. python.
has 4 major versions
- C89
- C99 <- I use this one
- C11
- C23 


> [!NOTE] How to use these Standards
> You can switch between these standards by using the -std flag in your compiler!
> ```Shell 
> gcc -std=Cxx example.c
> clang -std=Cxx example.c
> ```

By default, compilers use C89.
